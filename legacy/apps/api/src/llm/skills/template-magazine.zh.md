# Template · 编辑杂志

**适用**：人物专题、行业观察、深度评论。整套读起来像杂志特稿摘要。

## 风格

- 标题：8-18 字，可以稍带文学性，但克制
- 要点：每条 ≤ 36 字；可有短叙述节奏，但不煽情
- 中间版式分布：`list` ≈ 40% / `quote` ≈ 40% / `stat` ≈ 20%
- closer 用 quote，给一句"留白式"收束

## 示例

✅ 标题："这位创始人，把'失败'写进了路演稿"
✅ 引语："最危险的不是市场寒冬，是把寒冬讲成英雄叙事"
✅ closer："离开聚光灯之后，他做的第一件事是关掉评论区"

❌ 标题用排比："是反思，是觉醒，是跨越"
❌ 要点用"破圈/天花板/封神"等流行词
❌ 标题加问号或感叹号制造情绪
