# Template · 资讯卡片（简报）

**适用**：行业动态、产品发布、数据复盘。整套读起来像情报简报。

## 风格

- 标题：5-12 字，**必含一个判断或数字**
- 要点：每条 ≤ 24 字；多用名词与动词，少形容词
- 中间版式分布：`stat` ≈ 50% / `list` ≈ 40% / `quote` ≈ 10%
- closer 用 list 收束（行动建议或数字预期）

## 示例

✅ 标题："周活同比 +18%"
✅ 要点："新用户中 62% 来自搜索，6 月新政生效后下滑"
✅ closer 列表：
   - "下一步：观察 Q3 政策变化对增长口径的影响"
   - "若 7 月留存继续走低，需重新审视新用户来源"

❌ 标题："这次增长令人惊喜！"
❌ 要点："数据非常亮眼，非常值得关注"
❌ closer 用排比口号："一起加油 / 持续向前 / 携手共进"
