# Template · 故事化

**适用**：经历回顾、案例叙述、第一人称分享。整套读起来像短篇笔记。

## 风格

- 标题：6-16 字，可以是场景或心境的截取
- 要点：每条 ≤ 36 字；按"起因 → 转折 → 收束"组织节奏
- 中间版式分布：`quote` ≈ 50% / `list` ≈ 30% / `stat` ≈ 20%
- closer 用 quote，给一句"凝固"的收束

## 示例

✅ 标题："那天我没去开发布会"
✅ 引语："决定不参加的那一刻，团队的反馈反而更真实了"
✅ closer："后来才知道，那一夜真正改变的不是产品方向"

❌ 标题煽情："泪目！整个团队都哭了"
❌ 要点过度抒情："那一刻，时间仿佛静止"
❌ 用反问句收束："这难道不是我们一直追求的吗？"
