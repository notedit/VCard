# 卡组结构规则

整套卡组的版式分布与位置约束。这层约束硬性，不允许 AI 自行调整。

## 位置约束

- 第 1 张必须是 `cover`（封面）
- 最后 1 张必须是 `closer`（结尾）
- 中间各张从 `list / quote / stat` 里挑选

## 分布约束

- 不允许连续 3 张同类版式
- 当卡组数 ≥ 5 张时，至少包含 2 种中间版式（不能全是 list）
- `quote` 适合表达观点 / 引用；不适合放数据点
- `stat` 适合表达数字 / 数据 / 比例；不适合放纯文字论点

## 版式与内容的对应

- **cover**：标题 + 1-2 条总览级要点（提钩 / 提问题 / 提结论）
- **list**：标题 + 2-3 条平行要点（每条独立成立）
- **quote**：核心观点句 + 1-2 条解释性要点（核心句可作为 blockquote 显示）
- **stat**：1 个数字或比例 + 标题 + 1 条解释（数字本身就是论据）
- **closer**：收束 + 行动建议或下一步暗示（不是总结，是"留给读者一句话"）

## 必须

- 整套卡组遵循同一主题，不要中途偏题
- 如果主题包含时效性内容（最新发布、当年数据），先调用 web_search 工具 1-3 次取证再生成
