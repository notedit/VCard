# Anti-Slop · 中文社媒卡片黑名单

下列词、句式、标点用法在 vCard 任何 AI 输出中都不允许出现。生成完成后会做正则扫描；命中则触发一次重写，仍命中则在落库前替换/删除。

## A. 营销腔禁止词

破圈、出圈、入圈、内卷、躺平、赋能、抓手、闭环、链路、心智、红海、蓝海、护城河、降维打击、一文带你、深度好文、干货满满、家人们、姐妹们、宝子们

## B. 网络流行语禁止词

气质不符。一律避免。

绝绝子、yyds、i人e人、暴击、神来之笔、封神、王炸、天花板、救命、笑死、人麻了

## C. AI 标志性套句

LLM 高频生成、读起来像 AI 写的、必拉黑：

- "让我们一起来探索"
- "在这个数字化时代"
- "无独有偶"、"与此同时"、"不仅……还……"
- "不仅仅是 X，更是 Y"
- "总而言之"、"综上所述"
- "毫无疑问"、"众所周知"
- "在不久的将来"
- "随着……的发展"

## D. 句式与标点禁止

- 连续 ≥ 2 个感叹号（`！！`、`!!`）
- 反问开头（"你知道吗？"、"你以为 X？其实 Y"）
- emoji 超量：标题超过 1 个 emoji；要点超过 1 个 emoji
- 数字 + 百分号 + 夸张形容（"提升 200% 不是梦"、"暴涨 300%"）
- 标题用冒号双段结构（"X：让 Y 变得更 Z"）

## 检查规则

- 字符串包含匹配（A/B 类词不区分上下文，命中即违规）
- 套句模糊匹配：A 类整词、C 类近似（去标点 + 去空白后 includes）
- D 类用正则：`[！!]{2,}`、`你知道[吗么]？?`、`你以为.+其实`、`(\d+)%[^，。]{0,12}(暴|大|疯|不是梦|新高)`

## 重写优先

命中黑名单时，agent 应在保留原意前提下，**优先重写**为更克制的表达，而不是只删除关键词。例如：

- ❌ "Codex 让程序员破圈，YYDS！"
- ✅ "Codex 把开发者的工作从写代码扩展到委派任务"
