# Template · 极简专业

**适用**：资讯拆解、技术解释、观点输出。整套读起来像编辑部专栏。

## 风格

- 标题：5-14 字，给一个判断或动作动词；不堆形容词
- 要点：每条 ≤ 28 字；不用"首先 / 其次 / 再者"
- 中间版式分布：`list` ≈ 50% / `stat` ≈ 30% / `quote` ≈ 20%
- closer 用 quote 收束（一句话留给读者）

## 示例

✅ 标题："Codex 的能力边界在位移"
✅ 要点："从'给建议'转到'委派执行'，三类任务受影响"
✅ closer 引语："下一步不是更聪明的代码，是更清晰的任务拆分"

❌ 标题："让我们一起来探索 Codex 带来的全新可能"
❌ 要点："Codex 太牛了，简直是程序员的 YYDS"
❌ 标题用冒号双段："Codex：让程序员告别 996"
